Installing KD

Here's an introduction to install KD. We will first introduce the dependencies of KD, then we will present how to install KD through lines of command.

KDKA Requirements

Python dependency 

Python >= 3.7

Basic dependencies 

pytorch
numpy
scikit-learn
tqdm

KDKA Installation

Firstly, use git to clone the repository from GitHub to your local directory: 

$ git clone git@github.com:HiangX/KDKA.git

Then, cd to the KDKA directory and install the requirements through: 

$ cd KDKA
$ pip install -r requirements.txt

Then, execute following command to install KDKA: 

$ python setup.py install

Now, you can use KD in Python through a simple import: 

import KD

...

The KDKA method is also available within the KD package. See tutorial to learn how to use KD and KDKA.
